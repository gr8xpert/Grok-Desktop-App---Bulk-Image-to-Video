FFmpeg Setup for Meta Video Converter
=====================================

The Video Editor feature requires FFmpeg binaries.

Download and Setup:
1. Download FFmpeg from: https://www.gyan.dev/ffmpeg/builds/
   - Recommended: ffmpeg-release-essentials.zip

2. Extract the zip file

3. Copy the following files to this folder (assets/ffmpeg/):
   - ffmpeg.exe
   - ffprobe.exe

That's it! The Video Editor will now work with transitions, captions, and music.

Note: FFmpeg is a free and open-source software project.
Website: https://ffmpeg.org/
